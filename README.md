# Aptos Celebtron

Celebtron explain the transaction of Aptos chain

## Prequisites

- OpenAI API key
- Python >= 3.10

## How to run

1. Add `.env`

```dotenv
OPENAI_API_KEY=<your-openai-api-key>
```

or set ENV variable

```bash
export OPENAI_API_KEY=<your-openai-api-key>
```

2. Run api server

```bash
python server.py
```
